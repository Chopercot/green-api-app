# GREEN-API Тестовое задание (DevOps/Release Engineer)

Простое клиентское приложение для взаимодействия с WhatsApp через [GREEN-API](https://green-api.com/).

Проект контейнеризован с помощью Docker и раздается через Nginx.

2. Выполните команду в корне проекта:
